# dairy-milk-distributor-api-nodejs

A RESTful API for Dairy milk distributor to manage the milk distribution.

## Features

- Add Order
- Update Order
- Update Order Status
- Delete Order
- Check Today Left Milk Capacity
- Get All Orders
